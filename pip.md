% Pip

# Some basics

* to list outdated packages

```
	pip list --outdated
	print(s)
```
	
* to upgrade any outdated packages

```
	[Linux] pip install --upgrade [package_name]
	python -m pip install -U pip setuptools
```
	
* to upgrade pip

```
	python -m pip install --upgrade pip
```
	
* to list all installed packages

```
	pip list
```
